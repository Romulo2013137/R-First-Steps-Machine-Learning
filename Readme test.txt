Aqui foi usado o pacote caret para treinar um modelo de classificação Random Forest no conjunto de dados "Iris".

O pacote caret é muito flexível e pode lidar com todas essas tarefas.

Além disso, já que o projeto trabalha com conjuntos de dados maiores, futuramente irá usar métodos mais avançados para lidar com a maldição da dimensionalidade, lidar com características irrelevantes ou correlacionadas, e assim por diante.
